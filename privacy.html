Using Your Personal Content

When using the extension, some of your personal content will be used by the extension to perform basic functions!

These contents are only used on the client-side. The extension does not have a separate backend server. The extension can even completely disable all network permissions for offline use, so it will not collect any of your personal data.

Rules of Content Usage

Browsing Page Content

When using the extension, the extension will use your browsing page content for the following purposes:

  • Classify identification according to rules you set
  • Expand hidden content according to rules
  • Hide or remove content you do not want to see according to rules

Cookie

When using the extension, the extension will use Cookies for the following purposes:

  • Assist in identifying tweet content
  • Hide tweet replies
  • Hide users
  • Block users
  • Unblock users

Cookies are only used locally. The extension itself does not have the function to share or collect Cookies. Even when used locally, Cookies are managed and protected by the browser or operating system.

Github (Optional)

When using the extension, you can optionally use GitHub, and the extension will use GitHub for the following purposes:

  • Backup note information
  • Backup interception rule settings
  • Backup blacklists, whitelists
  • Share and exchange lists or settings with other users

The above purposes are based on your explicit settings. The extension itself will not automatically backup or share any data unless you explicitly set it!

If you choose to use GitHub, you should be aware of GitHub's privacy policy and related security recommendations!

Disclaimer

Security of Your Personal Data

The extension itself does not collect personal data. Since the extension does not have a dedicated backend server, it does not have the ability to collect personal data, and can even be used completely offline.

The extension does not use third-party services or components, so there are no concerns about third-party services or components leaking personal data.

However, please remember that browsers need the internet to function. As long as the internet is used, absolute security cannot be guaranteed. We can only ensure that the extension itself only uses your content locally, and the extension itself will not collect or send any of your personal data!
